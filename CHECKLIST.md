Lista funkcjonalności potrzebnych do wykonania przez oVirt Engine API
1.  [x] pobranie listy dostępnych Vnic profili
2.  [x] Pobranie listy konkretnych maszyn wirtualnych po frazie (np uniklany prefix dla każdego pod'a)
3.  [x] pobranie listy konkretnych maszyn wirtualnych po UUID
4.  [x] przypisanie Vnic profilu do interfejsu maszyny wirtualnej
5.  [x] odpięcie Vnic profilu od maszyny wirtualnej
6.  [x] dodanie interfejsu sieciowego maszynie wirtualnej
7.  [x] usunięcie interfejsu sieciowego maszynie wirtualnej
8.  [x] uruchomienie maszyny wirtualnej
9.  [x] zatrzymanie maszyny wirtualnej (shutdown)
10. [x] wyłączenie maszyny wirtualnej (power off)
